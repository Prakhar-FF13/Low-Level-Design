# Parking Lot LLD

## Problem Statement

Design a parking lot system that can handle multiple parking lots, each with multiple floors and multiple parking spots. The system should support the following features:

1. **Add a parking lot**: Add a new parking lot to the system.
2. **Add a floor**: Add a new floor to a parking lot.
3. **Add a parking spot**: Add a new parking spot to a floor.
4. **Park a vehicle**: Park a vehicle in a parking spot.
5. **Unpark a vehicle**: Unpark a vehicle from a parking spot.
6. **Get available spots**: Get the number of available spots in a parking lot.
7. **Get parked vehicles**: Get the list of parked vehicles in a parking lot.
